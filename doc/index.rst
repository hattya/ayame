Welcome to Ayame
================

Ayame is a component based WSGI framework. It is inspired by
`Apache Wicket`_, `Apache Click`_ and Flask_.

.. _Apache Wicket: http://wicket.apache.org/
.. _Apache Click: http://click.apache.org/
.. _Flask: http://flask.pocoo.org/
